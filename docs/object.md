# Object

### `isObjectEmpty(obj: object): boolean`

Function to check if an object is empty.

**Example:**

```typescript
let obj1 = {};
let obj2 = { key: "value" };
isObjectEmpty(obj1); // returns  true
isObjectEmpty(obj2); // returns  false
```
